
# Portfolio Template

Old portfolio kept as template.

## Features

- Responsive.
- English and French translation.
- Minifies css, javascript and optimize images with gulp.

## Requirements

- Node.js

## Configuration

Install dependencies
```
npm install
```
Install gulp command line
```
npm install --global gulp-cli
```
Minifies after modification (gulpfile.js)
```
gulp
```
